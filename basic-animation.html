Projects

  • Sprite Animation On A Background

    Assignment Focus

    This assignment introduced greater understanding of html and css and required the development of an animated character, coded using css, displayed on a personally designed background. The purpose of this assignment was to broaden my understanding of html, but specifically the capabilities of css and to demonstrate my understanding through character design and story telling techniques.

    Assigned Role

    For this assignment I undertook several tasks:

    • Researching inspiration to develop a unique character and story

    • Sketch several ideas, adopting inspiration from the gathered resources

    • Illustrate the final character and design a background suitable for the chosen storyboard

    • Programme the illustration, utilising appropriate CSS syntax and apply the animation sprite.

    Learning Outcome

    The learning outcome of this assignment includes gaining a greater understanding of:


    • Mopboards, and their ability to direct the focus towards specific design element. The use of the mood board allowed me to understand the colour, texture, common themes and overall perception of my chosen character style.

    • Animation principles were used within my assignment, specifically the pose-to-pose technique to ensure each frame of my character movement felt natural.

    • Illustrator abilities to design characters, whereby I utilised the trace feature to lock layers of my sketches onto various storyboards.

  • Javascript Controlled Animation

    Assignment Focus

    Utilising JavaScript, this assignment required our character to animate in various positions. The purpose of this assignment is to gain a greater understanding of JavaScript and its abilities to control the behaviour of an element.

    Assigned Role

    My role in this assignment was to develop my initial character, animated in css, however to utilise JavaScript statements in order to run particular animated actions.

    • Researching inspiration to develop a unique character and story

    • Sketch several ideas, adopting inspiration from the gathered resources

    • Illustrate the final character and design a background suitable for the chosen storyboard

    • Programme the illustration, utilising appropriate CSS syntax and apply the animation sprite.

    Learning Outcome

    I came to understand the use of variables and my ability to assign it a value which I can later use, with a query selector, for various elements. This is demonstrated through the use of sound in my animation buttons, to prompt the user about an action. Here I have assign a variable to a class which I can assign an event listener (function), which will allow me to activate the sound, depending on which action I would like the user to make.
